# Security Labs - 身份认证与攻击实验平台

这是一个包含6个安全实验的Web应用程序，用于演示身份认证技术及其相关攻击。

## 实验列表

1. **实验一：用户名/密码身份认证** - 简单的登录系统
2. **实验二：密码攻击** - 字典攻击模拟
3. **实验三：生物特征认证** - 基于面部识别的身份验证
4. **实验四：生物特征攻击** - 重放攻击模拟
5. **实验五：SSL握手协议** - SSL/TLS握手过程可视化
6. **实验六：SSL攻击** - 中间人攻击(MITM)模拟

## 安装与运行

### 1. 安装依赖

```bash
cd auth_security_lab
pip install -r requirements.txt
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

在浏览器中打开: `http://localhost:5000`

## 技术栈

- **后端**: Python Flask
- **前端**: HTML, CSS, JavaScript
- **认证**: 用户名/密码 + 生物特征(模拟)
- **安全协议**: SSL/TLS握手模拟

## 实验说明

### 实验一：用户名/密码认证
- 预设用户: `admin` / `password123`, `alice` / `wonderland`, `bob` / `builder`
- 演示基本的身份认证流程

### 实验二：字典攻击
- 针对实验一的登录系统进行暴力破解
- 使用预定义的密码字典

### 实验三：生物特征认证
- 使用摄像头捕获面部图像
- 模拟面部特征提取和验证

### 实验四：重放攻击
- 捕获并重放生物特征数据
- 演示生物识别系统的漏洞

### 实验五：SSL握手
- 可视化SSL/TLS握手的各个步骤
- 展示加密通道建立过程

### 实验六：SSL中间人攻击
- 模拟SSL降级攻击
- 演示弱加密协商的风险

## 注意事项

⚠️ **本项目仅用于教育目的，请勿用于非法用途**

- 生物特征认证使用简化的哈希比对，实际系统会使用更复杂的特征提取算法
- 密码以明文存储仅为演示，生产环境应使用加密存储
- SSL攻击为模拟演示，实际攻击需要更复杂的网络配置
